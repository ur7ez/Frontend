Что такое HTML?

HTML – язык гипертекстовой разметки для создания страниц в сети Internet.

Простейшая HTML-страница

Код простейшей html-страницы выглядит следующим образом:

<!DOCTYPE html>
<html>
  <head>
    <title>Я страничка!</title>
  </head>
  <body>
    <p>Привет, мир!</p>
  </body>
</html>
